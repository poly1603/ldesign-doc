# 国际化 (i18n)

LDoc 内置多语言支持，可以轻松构建多语言文档站点。

## 配置多语言

在 `doc.config.ts` 中配置 `locales` 选项：

```ts
export default defineConfig({
  lang: 'zh-CN',
  
  locales: {
    root: {
      label: '简体中文',
      lang: 'zh-CN'
    },
    en: {
      label: 'English',
      lang: 'en-US',
      link: '/en/',
      themeConfig: {
        nav: [
          { text: 'Guide', link: '/en/guide/' },
          { text: 'API', link: '/en/api/' }
        ]
      }
    }
  }
})
```

## 目录结构

多语言文档的目录结构：

```
docs/
├── index.md          # 默认语言首页
├── guide/
│   └── index.md
├── en/               # 英文版本
│   ├── index.md
│   └── guide/
│       └── index.md
└── ja/               # 日文版本（如需要）
    ├── index.md
    └── guide/
        └── index.md
```

## 语言切换器

配置多语言后，导航栏会自动显示语言切换下拉菜单。

## Locale 配置项

每个 locale 可以配置以下选项：

| 选项 | 类型 | 说明 |
|------|------|------|
| `label` | `string` | 语言显示名称 |
| `lang` | `string` | 语言代码 (BCP 47) |
| `link` | `string` | 语言路径前缀 |
| `themeConfig` | `object` | 该语言的主题配置覆盖 |

## 主题配置覆盖

每个语言可以覆盖主题配置中的任意选项：

```ts
locales: {
  en: {
    label: 'English',
    lang: 'en-US',
    link: '/en/',
    themeConfig: {
      // 覆盖导航栏
      nav: [
        { text: 'Guide', link: '/en/guide/' }
      ],
      // 覆盖侧边栏
      sidebar: {
        '/en/guide/': [
          { text: 'Introduction', link: '/en/guide/' }
        ]
      },
      // 覆盖页脚
      footer: {
        message: 'Released under the MIT License.'
      }
    }
  }
}
```

## 常用语言代码

| 语言 | 代码 |
|------|------|
| 简体中文 | `zh-CN` |
| 繁体中文 | `zh-TW` |
| English | `en-US` |
| 日本語 | `ja-JP` |
| 한국어 | `ko-KR` |
| Français | `fr-FR` |
| Deutsch | `de-DE` |
| Español | `es-ES` |

## 翻译 UI 文本

可以通过 `themeConfig.i18n` 覆盖内置 UI 文本：

```ts
themeConfig: {
  i18n: {
    search: 'Search',
    menu: 'Menu',
    toc: 'On this page',
    returnToTop: 'Return to top',
    appearance: 'Appearance',
    previous: 'Previous',
    next: 'Next',
    lastUpdated: 'Last updated'
  }
}
```

## 示例：中英双语站点

完整的双语配置示例：

```ts
import { defineConfig } from '@ldesign/doc'

export default defineConfig({
  title: 'My Docs',
  lang: 'zh-CN',
  
  locales: {
    root: {
      label: '简体中文',
      lang: 'zh-CN'
    },
    en: {
      label: 'English',
      lang: 'en-US',
      link: '/en/',
      themeConfig: {
        nav: [
          { text: 'Guide', link: '/en/guide/' },
          { text: 'API', link: '/en/api/' },
          { text: 'GitHub', link: 'https://github.com/...' }
        ],
        sidebar: {
          '/en/guide/': [
            {
              text: 'Getting Started',
              items: [
                { text: 'Introduction', link: '/en/guide/' },
                { text: 'Quick Start', link: '/en/guide/getting-started' }
              ]
            }
          ]
        },
        i18n: {
          search: 'Search',
          menu: 'Menu',
          toc: 'On this page'
        }
      }
    }
  },
  
  themeConfig: {
    nav: [
      { text: '指南', link: '/guide/' },
      { text: 'API', link: '/api/' },
      { text: 'GitHub', link: 'https://github.com/...' }
    ]
  }
})
```
