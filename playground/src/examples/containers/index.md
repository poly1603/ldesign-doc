# 容器与提示

LDoc 提供多种内置容器，用于突出显示重要信息。

## 信息容器

::: info 信息
这是一条普通信息提示，用于展示一般性说明。
:::

## 提示容器

::: tip 提示
这是一条提示信息，用于展示有用的建议。
:::

## 警告容器

::: warning 警告
这是一条警告信息，请注意相关内容。
:::

## 危险容器

::: danger 危险
这是一条危险警告，请务必小心处理。
:::

## 详情容器

::: details 点击展开详情
这里是隐藏的详细内容，点击标题可以展开或收起。

- 支持 Markdown 语法
- 可以包含代码块
- 可以嵌套其他内容
:::

## 自定义标题

::: tip 自定义提示标题
容器的标题可以自定义，只需在类型后面添加空格和标题文本即可。
:::

## 代码组

::: code-group

```bash [npm]
npm install ldoc
```

```bash [pnpm]
pnpm add ldoc
```

```bash [yarn]
yarn add ldoc
```

:::
