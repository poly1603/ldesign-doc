# Introduction

LDoc is a modern static documentation generator powered by Vite.

## Features

- **Blazing Fast** - Powered by Vite, instant HMR
- **Markdown Enhanced** - Code highlighting, containers, math formulas
- **Component Demo** - Native Vue/React component preview
- **Plugin Ecosystem** - Rich built-in plugins, easy to extend
- **TypeScript First** - Full type definitions

## Quick Start

```bash
# Create project
pnpm create ldoc my-docs

# Install dependencies
cd my-docs && pnpm install

# Start dev server
pnpm dev
```
