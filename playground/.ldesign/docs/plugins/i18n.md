# 国际化增强插件

`i18nPlugin` 提供：翻译状态追踪、翻译回退、RTL 布局支持。

## 安装

```ts
import { i18nPlugin } from '@ldesign/doc/plugins'

export default defineConfig({
  plugins: [
    i18nPlugin({
      statusTracking: { enabled: true },
      fallback: { enabled: true },
      rtl: { enabled: true }
    })
  ]
})
```

## 功能

- **翻译状态追踪**：构建结束时输出报告
- **翻译回退**：缺少翻译时使用回退语言，并可显示提示
- **RTL 支持**：对阿拉伯语等 RTL 语言注入布局样式

## 配置提示

在多语言站点中，建议配合 `locales` 与不同语言的 `themeConfig` 使用。
