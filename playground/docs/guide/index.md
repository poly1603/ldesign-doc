# 介绍

**LDoc** 是一个现代化的文档系统，灵感来源于 VitePress，但提供了更强大的功能和更灵活的扩展性。

## 为什么选择 LDoc？

LDoc 专为组件库和项目文档设计，提供了以下核心特性：

- **Vue 和 React 支持** - 在文档中直接渲染 Vue 或 React 组件演示
- **主题系统** - 完全可定制的主题，支持暗色模式
- **插件系统** - 通过插件扩展功能，如认证、搜索等
- **基于 Vite** - 极速的开发体验和构建性能

## 它是如何工作的？

LDoc 基于 Vite 构建，使用 markdown-it 处理 Markdown 文件，通过 Shiki 提供代码高亮。它将你的 Markdown 文件转换为 Vue 组件，并提供完整的路由和布局系统。

::: tip 提示
LDoc 设计为开箱即用，同时也支持深度定制。你可以使用默认主题快速开始，也可以创建自己的主题。
:::

| 功能 | VitePress | LDoc |

| 主题系统 | ✅ | ✅ |
