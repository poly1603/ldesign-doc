# Internationalization (i18n)

LDoc supports building multi-language documentation sites.

## Configuration

Configure languages in `doc.config.ts`:

```ts
export default defineConfig({
  // Default language
  lang: 'en-US',
  
  locales: {
    // Root path for default language
    root: {
      label: 'English',
      lang: 'en-US'
    },
    // Other languages
    zh: {
      label: '简体中文',
      lang: 'zh-CN',
      link: '/zh/'
    },
    ja: {
      label: '日本語',
      lang: 'ja-JP',
      link: '/ja/'
    }
  }
})
```

## Directory Structure

Organize translations in language directories:

```
docs/
├── index.md          # English (root)
├── guide/
│   └── index.md
├── zh/               # Chinese
│   ├── index.md
│   └── guide/
│       └── index.md
└── ja/               # Japanese
    ├── index.md
    └── guide/
        └── index.md
```

## Language-specific Theme Config

Override theme config per language:

```ts
export default defineConfig({
  locales: {
    root: {
      label: 'English',
      lang: 'en-US'
    },
    zh: {
      label: '简体中文',
      lang: 'zh-CN',
      link: '/zh/',
      themeConfig: {
        nav: [
          { text: '指南', link: '/zh/guide/' },
          { text: 'API', link: '/zh/api/' }
        ],
        sidebar: {
          '/zh/guide/': [
            {
              text: '开始使用',
              items: [
                { text: '介绍', link: '/zh/guide/' }
              ]
            }
          ]
        }
      }
    }
  }
})
```

## Language Switcher

The language switcher automatically appears in the navigation bar when multiple languages are configured.

Users can click the globe icon to switch between available languages.

## Translation Tips

1. **Keep URLs consistent** - Use the same file structure across languages
2. **Translate frontmatter** - Update title, description for each language
3. **Handle missing translations** - Fallback to default language for missing pages
4. **Use translation keys** - For UI elements, consider using translation utilities

## RTL Support

For right-to-left languages:

```ts
locales: {
  ar: {
    label: 'العربية',
    lang: 'ar-SA',
    link: '/ar/',
    dir: 'rtl'
  }
}
```
