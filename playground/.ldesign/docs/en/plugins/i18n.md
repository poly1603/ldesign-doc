# I18n Enhancements Plugin

`i18nPlugin` provides translation status tracking, translation fallback and RTL layout support.

## Installation

```ts
import { i18nPlugin } from '@ldesign/doc/plugins'

export default defineConfig({
  plugins: [
    i18nPlugin({
      statusTracking: { enabled: true },
      fallback: { enabled: true },
      rtl: { enabled: true }
    })
  ]
})
```
